# Portfolio

Hello, World!
My name is Bao.
Welcome to my personal portfolio.
Powered by React + Tailwind and ASP.NET Core C#.

## Repo structure

```
portfolio/
├── frontend/       # React + Tailwind CSS
├── backend/        # ASP.NET Core Web API (C#)
└── .github/
    └── workflows/  # CI/CD pipelines
```

## Branches

| Branch | Purpose |
|---|---|
| `main` | Production |
| `dev` | Integration |
| `feature/*` | Feature branches |

## Getting Started

```bash
# Frontend
cd frontend
npm install
npm run dev

# Backend
cd backend
dotnet restore
dotnet run
```
